Chunk Testing Framework

Our comprehensive testing framework evaluates different chunking strategies to optimize retrieval performance for legislative documents and civic data.

{/* Chunking Strategies */}

Chunking Strategies

Plank-Structured

Preserves document hierarchy with custom PDF preprocessing for political platforms and resolutions.

Paragraph-Based

Splits content by paragraph boundaries for natural semantic groupings.

N-Sentence Chunking

Groups content by sentence count (3-sentence default) for consistent context windows.

Token-Based

Fixed token limits using BERT tokenization for embedding model compatibility.

{/* Evaluation Metrics */}

Evaluation Metrics

Chunk Recall

Measures how well the system retrieves relevant document chunks for given queries.

Answer F1 Score

Evaluates precision and recall of generated answers against ground truth responses.

Faithfulness

Ensures generated responses remain grounded in retrieved source material.

Execution Time

Tracks processing speed for real-time performance optimization.
